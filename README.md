# 🩸 Rakht Seva

> A Blood Donation Management System that connects blood donors, patients, and hospitals in one platform.

---

# 📌 Overview

Rakht Seva is a web-based application designed to help people quickly find blood donors during emergencies. The platform allows users to register as donors, search for blood groups, request blood, and manage donor information efficiently.

This project aims to make the blood donation process faster, easier, and more accessible.

---

# 🚀 Features

✅ User Registration & Login  
✅ Blood Donor Management  
✅ Search Donors by Blood Group  
✅ Blood Request System  
✅ Hospital/Patient Information  
✅ Responsive UI Design  
✅ Database Integration

✅ Contact & Emergency Support
